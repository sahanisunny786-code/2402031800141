# Neon Weather App

## Overview
The Neon Weather App provides users with accurate and real-time weather information. With an intuitive and vibrant user interface, it helps you stay updated with the latest weather conditions in your area.

## Features
- **Real-time Weather Data**: Get updated weather data for any location.
- **User-friendly Interface**: Designed for ease of use with colorful and engaging visuals.
- **Forecasts**: Check hourly and daily forecasts for better planning.
- **Search Functionality**: Easily search for any location.
- **Responsive Design**: Works seamlessly across various devices.

## Setup Instructions
1. **Clone the Repository**: 
   ```bash
   git clone https://github.com/sahanisunny786-code/2402031800141.git
   cd 2402031800141
   ```
2. **Install Dependencies**: 
   ```bash
   npm install
   ```
3. **Run the App**: 
   ```bash
   npm start
   ```
4. **Open in Browser**: Navigate to `http://localhost:3000` to view the app.
